# 馕言文规则

## 核心

保留原意，改成倒装、放飞比喻、带点直译骨架的馕言文。

## 常用手法

- 调整语序，先说结果或态度。
- 加少量口头词，不要每句都塞满。
- 句首优先用称呼开场，像在跟人当面说话。
- 用馕、烤包子、烤肉、风、沙、天边一类意象做替换。
- 把抽象情绪改成具体画面，比如饿 -> 空仓库，怒 -> 没空气处理，笨 -> 杆子拿着也整不明白。
- 短句也必须有至少一个感官画面，比如嘴巴发烫、脑袋发晕、肚子空、手发抖。
- 先保留原句骨架，再把词换成奇特比喻。
- 几乎不要用“像”。比喻优先写成“X一样的Y有的呢”，比如“雄鹰豹子一样的速度有的呢”。
- 优先写成“主体 + 比喻 + 有的呢/得很”，别拆开说。
- 例如“你家里这个鹦鹉巴掌大的小脑袋装着绵羊一样的安静有的呢”。
- 比喻要紧贴主体，别在后面再补解释尾巴。
- 结尾要短，最好一击就收。
- `第一`、`最好`、`很棒` 这类评价词，优先收进比喻里，不要单独落地。
- 负面句里，尽量把 `不想`、`不会`、`很差` 直接压成一个画面，不要再保留第二层判断。
- 夸张一点，但别把意思改跑。
- 保留一种“哎朋友，我跟你说”那股劲儿。

## 不要做

- 不要把中文变成胡乱拼接的方言。
- 不要强行堆砌新疆词。
- 不要用普通话式“像……一样……”比喻。
- 不要改掉数字、人名、专有名词。
